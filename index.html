O que é HTML

(Voltar)

Ao contrário do que muitos pensam, o html não é uma linguagem de programação, e sim uma linguagem de marcação de texto, como seu próprio nome diz (HyperText Markup Language).
Além disso, ela é a estrutura base para a criação de uma página web que pode ser interpretada por navegadores, desde um simples bloco de notas até softwares especificos para a utilização do mesmo. Essa estrutura é formada por um conjunto de elementos, ou seja, os hipertextos*, que se conectam entre si formando a página.

O que são Tags

(Voltar)

Tags nada mais são do que códigos utilizados dentro do html, que servem como comandos que ajudam a formar a estrutura da página que será criada, sem que a mesma seja visível no browser*.
Por sua vez, são utilizados para informar ao navegador que tipo de estrutura está sendo construída, podendo conter títulos, parágrafos, imagens, links, entre outros.
Geralmente são representadas por uma estrutura própria, onde iniciam com o sinal “menor que”, em seguida vem o nome de um elemento e termina com o sinal “maior que”.
O fechamento de uma tag é feito com uma barra (/), sendo que no caso das tags de auto fechamento, não há necessidade da presença desse caractere (self-closing).
As tags também podem ser categorizadas inicialmente em dois tipos, em “nível de bloco” (block-level) e “em linha” (inline).

Principais Tags e Elementos

(Voltar)

Existem vários tipos de tags HTML, sendo que cada uma tem um fim específico


Algumas delas são:
  • Tags de cabeçalho: estão relacionadas com o cabeçalho e alguns elementos básicos de sites, como o título por exemplo;

  • Tags de links: são utilizadas para gerenciar qualquer tipo de link existente na página;

  • Tags de listas: permitem criar diferentes listas, sendo elas ordenadas e não ordenadas;

  • Tags de conteúdo multimídia: utilizadas para inserir elementos como imagens, vídeos e músicas nas páginas web;

  • Tags de formulários: utilizadas para criar formulários para os usuários;

Todas as páginas HTML possuem uma série de elementos, que consistem num conjunto de tags e atributos. Sendo assim, qualquer objeto identificável dentro de um documento como, por exemplo, um caractere, palavra, imagem, parágrafo ou célula de planilha.

Alguns deles são:
  • Parágrafo: (< p >)
  • Cabeçalho: (< h1 > ao < h6 >)
  • Imagem: (< img >)
  • Âncora: (< a >)
  • Lista ordenada: (< ol >)
  • Lista não ordenada: (< ul >)
